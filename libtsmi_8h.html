Detailed Description

Realtime roguelike engine.

libtsmi is an engine for realtime, tile-based, text-graphic RPG games. It is written in C99, and makes use of the libtcod roguelike library, and the GLib general-purpose utility library.

This is being developed in conjunction with a game I am writing in Ruby, so an API that is easy to use in higher level languages is a priority.

Creates a 2D binary space parition tree. This is a recursive function, and will continue until the nodes are too small to statisy the min_height and min_width constraints.

Parameters:
parent Root node of the tree to be created. Should span the whole area.
min_width minimum width of a child node
min_height minimum height of a child node
leaves A linked list of all the leaves, or nodes without children. This should grow as the function recurses. NOTE that since the basic GList* is a pointer data structure, and the GList* is what gets modified (not the GList struct itself), a pointer to a pointer GList** is what gets passed in.
